# -resumo-do-lab
Este repositório contém o resumo das lições aprendidas durante o desenvolvimento do lab na DIO"

O vídeo apresenta um laboratório introdutório de computação em nuvem, explicando os conceitos básicos e mostrando, de forma prática, como esse modelo funciona.
São abordadas ideias como o uso de recursos remotos via internet, virtualização, escalabilidade e os principais modelos de serviço (como infraestrutura, plataforma e software como serviço). 
Ao longo do laboratório, o instrutor demonstra como acessar e utilizar um ambiente em nuvem, destacando vantagens como redução de custos, flexibilidade e facilidade de gerenciamento 
em comparação com servidores locais, reforçando a importância da computação em nuvem no contexto atual da tecnologia.
